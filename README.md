## Usage example

```dart
import 'package:stepper_touch/stepper_touch.dart';
...
Container(
  padding: const EdgeInsets.all(8.0),
  child: StepperTouch(
    initialValue: 0,
    direction: Axis.vertical,
    withSpring: false,
    onChanged: (int value) => print('new value $value'),
  ),
),
...
```
